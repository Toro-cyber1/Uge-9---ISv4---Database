## App der viser et simpelt lager/ordre-system og styrer en UR-robot og og gemmer data i en SQLite-database.
- Start URSim + app.

- Vis 2 ordrer i Queued Orders.

- Klik “Process next order ➜” to gange.

## Vis:

  - Ordrer flytter til Processed Orders

  - Total revenue stiger

  - Robotten bevæger sig i URSim.
